## ⚠️ Důležité upozornění

Tyto dokumenty jsou **nezávazné šablony**. Před použitím doporučuji:
1. Konzultovat s právníkem / advokátem
2. Upravit podle konkrétní situace (práce přes živnost × s.r.o.)
3. Zvážit:
   - **Smlouvu o dílo** místo smlouvy o spolupráci (často lepší pro IT projekty)
   - **NDA** pokud řešíte citlivé know-how
   - **Licenční ujednání** ke zdrojovým kódům
